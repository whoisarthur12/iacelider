APara los fines del abordaje de los trabajo de INTERPOL, lo términos enlistados
a continuación deben entenderse de la siguiente manera:
Información criminal: una serie de datos como información
```
sociodemográfica sobre delincuentes (edad, sexo, tipo de empleo, etc.), y
```
los tiempos y ubicaciones de la actividad delictiva. Es clave para
comprender el funcionamiento interno y los factores impulsores de los
```
fenómenos y las actividades delictivas (Organización Internacional de
```
```
Policía Criminal, s.f.).
```
Perfil criminológico: técnica de apoyo a la investigación criminal, que
pretende conocer las características, motivaciones y actuaciones del autor
```
de un delito (por lo general culpable de homicidios y/o violaciones
```
```
seriales), a partir del análisis y evaluación de la evidencia física y
```
psicológica que deja el agresor en la escena del crimen y la víctima
```
(Céspedes et al., 2013).
```
Modus operandi: describe la forma en que se ejerció la conducta punible,
```
permitiendo conocer otras variables como, el tipo de víctima (conocido-
```
```
desconocido), uso de armas (blanca, de fuego u otras), lugar en el que se
```
```
cometió el delito y el contexto general de los hechos (Marcela, n.d.).
```
```
Arresto: Restricción efectiva de libertad con el propósito de responder por
```
la comisión de un hecho punible en la forma y manera establecidas por ley
```
(Suprema Corte de Justicia, 2021).
```
```
Interrogatorio: es un proceso que persigue conocer los detalles de la
```
comisión de un delito, obtener una confesión o datos incriminatorios
```
suficientes para enjuiciar al autor (UNIR, 2021).
```
Formulación precisa de cargos: principio rector del proceso penal que
dicta que “toda persona tiene el derecho de ser informada previa y
detalladamente de las imputaciones o acusaciones formuladas en su
```
contra” (Congreso Nacional, 2015, p.11).
```
13
Extradición: es un instituto del Derecho Internacional Público de aplicación
```
en el derecho criminal; mediante este instituto las autoridades judiciales de
```
un país solicitan la entrega de un encartado, procesado o imputado o
sospechoso a las autoridades de otro quien a su vez dispone los medios
necesarios para entregar al encartado o procesado, imputado o
sospechoso o en cuestión al solo efecto de proseguir con el proceso
```
(Garay, 2003).
```
Notificaciones rojas: Para localizar y detener a personas buscadas con
miras a su enjuiciamiento o para cumplir condena.
Notificaciones amarillas: Para localizar a personas desaparecidas,
frecuentemente menores, o ayudar a identificar a personas incapaces de
identificarse a sí mismas.
Notificaciones azules: Para conseguir más información sobre la identidad
de una persona, su paradero o sus actividades delictivas en relación con
una investigación penal.
Notificaciones negras: Para conseguir información sobre cadáveres por
identificar.
Notificaciones verdes: Para alertar sobre las actividades delictivas de una
persona, si se considera que esta comporta un peligro para la seguridad
pública.
Notificaciones naranjas: Para alertar acerca de un acontecimiento, una
persona, un objeto o un procedimiento que supongan un peligro grave e
inminente para la seguridad pública.
Notificaciones moradas: Para buscar o facilitar información sobre modus
operandi, objetos, dispositivos y métodos de ocultación utilizados por los
delincuentes.
Notificaciones especiales de INTERPOL y el Consejo de Seguridad de las
Naciones Unidas: Para informar sobre entidades y personas que son objeto
de sanciones impuestas por los Comités de Sanciones del Consejo de
Seguridad de las Naciones Unidas.

protagonizado un “grupo delictivo” compuesto por varias personas que se

actúan de manera concertada con el propósito de cometer uno o más

Historia
18
Si bien la INTERPOL se estableció oficialmente en 1923, sus raíces se remontan a
un encuentro anterior. En abril de 1914, Mónaco fue el escenario del primer
Congreso Internacional de Policía Criminal, un evento crucial que sentó las
```
bases para la futura cooperación policial internacional (Organización
```
```
Internacional de Policía Criminal, s.f.).
```
Convocados por el Príncipe Alberto I de Mónaco, abogados y funcionarios
policiales de 24 países se congregaron para abordar temas de vital
```
importancia: la búsqueda de soluciones conjuntas para combatir el crimen, el
```
intercambio de técnicas de identificación y la definición de procedimientos de
extradición.
El congreso de 1914 marcó un hito en la historia de la cooperación policial
internacional. Sin embargo, el estallido de la Primera Guerra Mundial truncó
abruptamente el desarrollo de este plan tan prometedor.
A pesar de este revés, la semilla sembrada en Mónaco en 1914 germinaría años
más tarde, dando vida a INTERPOL, una organización que hoy en día es
fundamental para la seguridad global.
Tras el paréntesis forzoso de la Primera Guerra Mundial, la idea de crear un
organismo internacional de policía resurgió con ímpetu. La figura clave en este
resurgimiento fue Johannes Schober, quien ostentaba el cargo de Presidente
de la Policía de Viena.
Convencido de la necesidad de una cooperación policial global para enfrentar
el crimen transnacional, Schober tomó la iniciativa. En 1923, convocó el
```
segundo Congreso Internacional de Policía Criminal en Viena (Austria). Este
```
```
evento crucial reunió a representantes de 20 países (Organización
```
```
Internacional de Policía Criminal, s.f.), quienes se congregaron para reavivar la
```
visión de una organización policial internacional, estos fueron:
Alemania Austria Checoslovaquia Dinamarca Egipto
EE.UU. Fiume Francia Hungría Italia
Japón Letonia Países Bajos Polonia Rumania
Grecia Suecia Siuza Turquía Yugoslavia
El Congreso de Viena marcó un punto de inflexión fundamental. Las ideas y
propuestas discutidas durante este encuentro sentaron las bases para la
creación oficial de INTERPOL solo unos meses más tarde. De este modo, la
determinación y el liderazgo de Johannes Schober fueron pilares esenciales en
el nacimiento de esta organización policial de vital importancia para la
seguridad global.
El 7 de septiembre de 1923, un nuevo capítulo se escribió en la historia de la
cooperación policial internacional. En esa fecha, durante una reunión crucial,
se tomó la decisión de crear la Comisión Internacional de Policía Criminal
```
(CIPC).
```
El objetivo primordial de la CIPC era establecer un marco para la asistencia
recíproca entre las fuerzas policiales de diferentes países. Con la creación de
esta organización, se buscaba enfrentar de manera conjunta los desafíos del
```
crimen transnacional y fortalecer la seguridad global (Organización
```
```
Internacional de Policía Criminal, s.f.)
```
1920
Los fines y la estructura de la CIPC quedaron plasmados en una serie de
resoluciones, sentando las bases para su funcionamiento y estableciendo los
principios rectores de la cooperación policial internacional.
Entre los temas que centraron la atención de la CIPC se encontraban:
Contacto policial directo: Facilitar la comunicación directa entre las
fuerzas policiales de distintos países para agilizar el intercambio de
información y la coordinación de acciones.
Cooperación en detenciones y extradición: Establecer mecanismos
eficientes para la detención y extradición de criminales que operaban a
nivel transnacional.
Idiomas comunes: Promover el uso de idiomas comunes para facilitar la
comunicación y colaboración entre las fuerzas policiales de diferentes
países.
Oficinas especializadas: Crear oficinas especializadas para abordar
delitos específicos, como la falsificación de moneda, cheques y
pasaportes.
Técnicas de identificación: Estandarizar y optimizar las técnicas de toma
de huellas dactilares y el registro de estas, para mejorar la identificación de
criminales.
La creación de la CIPC marcó un hito fundamental en la historia de la
cooperación policial internacional. Esta organización sentó las bases para el
desarrollo de INTERPOL, la organización policial internacional que hoy en día

Objetivos Policiales Mundiales
22
INTERPOL, en su constante compromiso con la seguridad global, ha establecido
siete Objetivos Policiales Mundiales. Estos objetivos representan una hoja de
ruta ambiciosa y colectiva para que la comunidad policial internacional
trabaje en conjunto hacia un mundo más seguro.
En consonancia con la Agenda 2030 de las Naciones Unidas para el Desarrollo
Sostenible, los Objetivos Policiales Mundiales se basan en una visión integral
que reconoce la interconexión entre la seguridad y el desarrollo sostenible.
Aprobados por primera vez en 2017, estos objetivos han sido revisados en 2023
para adaptarse a la rápida evolución del panorama de seguridad, impulsado
```
por los avances tecnológicos (Organización Internacional de Policía Criminal,
```
```
2023)
```
Cada uno de los siete Objetivos Policiales Mundiales se centra en un área
delictiva específica o una amenaza global:
Objetivo nº 1: Permitir a la comunidad mundial de las fuerzas del orden
combatir y prevenir con mayor eficacia el terrorismo mediante la
cooperación internacional.
Objetivo nº 2: Promover la seguridad fronteriza a escala mundial.
Objetivo nº 3: Mejorar la respuesta policial para proteger a los grupos
vulnerables.
Objetivo nº 4: Reducir el daño y la repercusión de la ciberdelincuencia a
escala mundial.
Objetivo nº 5: Combatir la corrupción y los delitos financieros en todas sus
formas.
Objetivo nº 6: Combatir la delincuencia organizada y el tráfico de drogas.
Objetivo nº 7: Reforzar la seguridad medioambiental y apoyar el fomento
de medios de subsistencia sostenibles combatiendo los delitos que
afectan al medio ambiente y al clima.
23
Los Objetivos Policiales Mundiales son transversales, lo que significa que cada
uno de ellos está interconectado con los demás. La implementación exitosa de
un objetivo requiere la consideración y el progreso en los demás.
INTERPOL, como organización líder en la cooperación policial internacional,
juega un papel crucial en la promoción y el logro de estos objetivos. La
organización facilita la colaboración entre las fuerzas del orden de todo el
mundo, brinda herramientas y recursos especializados, y promueve el
intercambio de información y mejores prácticas.
Los Objetivos Policiales Mundiales representan un compromiso colectivo de la
comunidad internacional para crear un mundo más seguro. Al trabajar juntos,
las fuerzas del orden pueden hacer frente a las amenazas globales de manera
efectiva y construir un futuro más pacífico y justo para todos.
